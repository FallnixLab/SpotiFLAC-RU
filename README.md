# SpotiFLAC-RU

<a href="https://trendshift.io/repositories/15737" target="_blank"><img src="https://trendshift.io/api/badge/repositories/15737" alt="afkarxyz%2FSpotiFLAC | Trendshift" style="width: 250px; height: 55px;" width="250" height="55"/></a>

**SpotiFLAC-RU** — это русская локализация инструмента для загрузки треков Spotify в настоящем качестве FLAC из Tidal, Qobuz и Amazon Music. Аккаунт не требуется.

---

### 🇷🇺 Особенности этой версии (RU Fork):
- **Полный перевод интерфейса:** Все меню, настройки и сообщения теперь на русском языке.
- **Локализованные уведомления:** Тексты ошибок и прогресса загрузки полностью переведены.
- **Оптимизированная сборка:** Исправлены ошибки зависимостей для корректной компиляции на Windows.

## Часто задаваемые вопросы (FAQ)

<details>
<summary>Эта программа бесплатна?</summary>

_Да. Программа полностью бесплатна. Вам не нужен аккаунт, подписка или регистрация. Все, что требуется — подключение к интернету._

</details>

<details>
<summary>Могут ли мой аккаунт Spotify забанить за использование этой программы?</summary>

_Нет. Программа никак не связана с вашим аккаунтом. Данные Spotify получаются путем обратной разработки веб-плеера, а не через вашу авторизацию._

</details>

<details>
<summary>Откуда берется аудио?</summary>

_Аудио загружается с использованием сторонних API (Tidal, Qobuz, Amazon и др.)._

</details>

<details>
<summary>Почему иногда не удается загрузить метаданные?</summary>

_Обычно это происходит из-за временного ограничения вашего IP-адреса сервисом. Подождите немного или используйте VPN._

</details>

<details>
<summary>Почему антивирус или Windows Defender ругается на файл?</summary>

_Это ложное срабатывание. Скорее всего, это связано с тем, что исполняемый файл сжат с помощью UPX. Если вы беспокоитесь, вы можете самостоятельно собрать программу из исходного кода._

</details>

<summary>Хотите поддержать проект автора?</summary>
